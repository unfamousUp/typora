将jar包上传到本地仓库

```cmd
mvn install:install-file -DgroupId=com.techotaku -DartifactId=ucb -Dversion=1.0-SNAPSHOT -Dpackaging=jar -Dfile=G:/code/github/cs61b/skeleton-sp21/proj0/javalib/ucb.jar
```

仓库路径前需要加file:///表示本地文件系统

```
mvn deploy:deploy-file -DgroupId=com.techotaku -DartifactId=hamcrest-core -Dversion=1.0-SNAPSHOT -Dpackaging=jar -Dfile=G:/code/github/cs61b/skeleton-sp21/proj0/javalib/hamcrest-core-1.3.jar -Durl=file:///E:/software/apache-maven-3.6.3/mvn_resp
```
