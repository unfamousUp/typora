### 2. Node.js简介

#### 2.1 什么是Node.js

> - `Node.js是一个`基于Chrome V8引擎的`JavaScript运行环境`
>   - 前端运行环境：浏览器
>   - 后端运行环境：Node.js
> - 官方网站：https://nodejs.org/zh-cn/
